Propósito del Sitio

Por este medio se crea una interfaz web donde se refleja el contenido de esta usando el lenguaje de programación HTML con el Framework CSS para diseños elegantes y dinámicos.

Introducción

CSS es el acrónimo de Cascading Style Sheets, lo que sería en español hojas de estilos en cascada. Es un lenguaje que sirve para especificar el estilo o aspecto de las páginas web. CSS se define en base a un estándar publicado por una organización llamada W3C, que también se encarga de estandarizar el propio lenguaje HTML.


El lenguaje HTML está limitado a la hora de aplicar la forma aun documento. Sirve de manera excelente para especificar un contenido que debe tener una página web, pero no permite definir como ese documento que se debe presentar.


Al principio para solucionar las limitaciones de este lenguaje, los diseñadores usaron técnicas como las tablas, imágenes, márgenes, etiquetas. Aunque daban sus resultados estéticos, lo cierto es que causaban muchos problemas, en la hora de mantener la visualización en diferentes navegadores.


Finalmente, el motivo de la necesaria creación de CSS de maquetar en forma dinámica el documento en HTML, Sin duda fue un paso adelante muy importante, pero la web siguió avanzando a un ritmo más rápido que los estándares y los diseñadores seguían teniendo que arreglárselas para conseguir efectos que eran imposibles con HTML y CSS.
